## 구현 기능

<br>

### 사용자

- 로그인
- ID 찾기
- PW 찾기
- 회원 가입
- 마이 페이지
  - 예약 현황 내역
    - 예약 현황 상세
  - 마일리지
  - 후기 내역
  - 문의 내역


<br>

### 사용자 - 오피스

- 헤더 - 사용자 메뉴 / 푸터
  - 헤더 - 서치바
- 메인 화면
- 공간 리스트 화면
- 공간 상세 페이지
  - 공간 상세 (공간 소개)
  - 공간 상세 (문의 사항)
    - 공간 상세 (문의 작성 팝업)
  - 공간 상세 (후기)
    - 공간 상세 (후기 작성 팝업)
- 공간 예약 Picker
- 공간 예약 및 결제

<br>

### 백오피스
- 메인 화면 (헤더/푸터)
- 로그인
- 비밀번호 찾기
- Host 신청
- Dash Board Home
- 공간 관리 (리스트)
- 공간 관리 (추가 / 수정)
- 공간 관리 (문의)
- 공간 관리 (문의 답변)
- 예약 관리
- 정산
- 환경 설정
  - 업체 삭제 요청
  - 로그아웃

<br>
